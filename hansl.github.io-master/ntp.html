个人简介

查看文件#cat /etc/ntp.conf

内容如下：

driftfile /var/lib/ntp/drift

pidfile /var/run/ntpd.pid

#logfile /var/log/ntp.log

# Access Control Support

restrict default

restrict -6 default

restrict 127.0.0.1

# local clock

server 127.127.1.0

fudge 127.127.1.0 stratum 3

logfile /var/log/ntp.log

server time1.aliyun.com iburst minpoll 4 maxpoll 6 prefer

server time2.aliyun.com iburst minpoll 4 maxpoll 6

公共时钟源：

Unix类系统：time1-7.aliyun.com

Windows： time.pool.aliyun.com

手动同步时钟源

停止ntpd

systemctl stop ntpd.service

手动同步

ntpdate 时钟源域名或IP地址

重启ntpd服务

systemctl restart ntpd.service
